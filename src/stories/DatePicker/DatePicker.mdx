# Date Picker

<div className="tip-wrapper">

A flexible and customizable date picker component for modern web applications.

## Overview

The Date Picker provides an intuitive and feature-rich interface for date selection. It's built as a lightweight, framework-agnostic web component that works in any modern web application. The component supports single date selection, date range selection, localization, theming, event indicators, and custom input elements.

### Key Features

- **Single and Range Selection**: Select either a single date or a date range with clear visual feedback
- **Multiple Calendar Views**: Day, month, and year selection views with smooth transitions
- **Direct Text Input**: Type dates directly into the input field with format validation
- **Custom Calendar Triggering**: Hide the default calendar icon and use your own trigger elements
- **Date Restrictions**: Disable specific dates, date ranges, weekdays, or set min/max boundaries
- **Event Indicators**: Mark dates with customizable event indicators and tooltips
- **Range Selection with Available Dates**: Get a list of valid dates in a range, excluding disabled ones
- **Internationalization**: Support for multiple languages, date formats and first day of week preferences
- **Theme Customization**: Extensive styling options via CSS custom properties
- **Responsive Design**: Optimized for desktop and mobile with touch gestures support
- **Keyboard Accessibility**: Full keyboard navigation and screen reader support
- **Animation Effects**: Smooth transitions between months and calendar views
- **Slotable Input**: Use custom input elements while maintaining calendar functionality.

## Component Architecture

The Date Picker follows a modular architecture with several key services:

- **UIService**: Handles DOM rendering and UI updates
- **StateService**: Manages component state and notifies listeners of changes
- **DateFormatter**: Handles date formatting and parsing according to locales
- **KeyboardNavigationService**: Manages keyboard navigation and accessibility
- **CalendarService**: Handles calendar calculations and date utilities

The component uses several view classes for different parts of the UI:
- **CalendarView**: Renders the day grid view
- **MonthView**: Renders the month selection view
- **YearView**: Renders the year selection view
- **HeaderView**: Renders the header with navigation controls
- **FooterView**: Renders the action buttons (Today, Clear, Close)

| `dd` | Day with leading zero (01-31) | 07 |

## Keyboard Navigation

The Date Picker component offers full keyboard support for accessibility:

### When Calendar is Closed
- `Tab`: Focus on the input field or custom input
- `Enter` or `Space` when focused on input: Open the calendar
- `Escape`: If calendar is open, close it

### When Calendar is Open (Day View)
- `Arrow Keys`: Navigate between days
- `Page Up`: Go to previous month
- `Page Down`: Go to next month
- `Home`: Go to the first day of the current month
- `End`: Go to the last day of the current month
- `Enter` or `Space`: Select the focused date
- `M`: Switch to month selection view
- `Y`: Switch to year selection view
- `Escape`: Close the calendar
- `T`: Navigate to today's date

### Month Selection View
- `Arrow Keys`: Navigate between months
- `Enter` or `Space`: Select the focused month and return to day view
- `Escape`: Return to day view without selection

### Year Selection View
- `Arrow Keys`: Navigate between years
- `Page Up`: Go to previous year range
- `Page Down`: Go to next year range
- `Enter` or `Space`: Select the focused year and go to month view
- `Escape`: Return to previous view without selection

## JavaScript API

### Properties and Methods

#### Getting and Setting Dates

```javascript
const datePicker = document.querySelector('odyssey-date-picker');

// Get the selected date
const selectedDate = datePicker.getDate();

// Set a date programmatically
datePicker.setDate(new Date(2025, 3, 22)); // April 22, 2025

// For range mode
const rangePicker = document.querySelector('odyssey-date-picker[mode="range"]');
const range = rangePicker.getDateRange();
console.log(range.startDate, range.endDate);

// Set a date range programmatically
rangePicker.setDateRange(
  new Date(2025, 3, 15), // April 15, 2025
  new Date(2025, 3, 22)  // April 22, 2025
);

// Get all available dates in the range (excluding disabled dates)
const availableDates = rangePicker.getAvailableDatesInRange();
console.log(`Available dates in range: ${availableDates.length}`);
```

#### Working with Events

```javascript
// Add event indicators
datePicker.addEvent(new Date(2025, 3, 15), 'Meeting');
datePicker.addEvent(new Date(2025, 3, 22), 'Deadline');

// Clear events
datePicker.clearEvents(new Date(2025, 3, 15));
```

#### Managing Disabled Dates

```javascript
// Disable specific dates (like public holidays)
datePicker.addDisabledDate(new Date(2025, 11, 25), "Christmas Day");
datePicker.addDisabledDate(new Date(2025, 0, 1), "New Year's Day");

// Disable multiple dates at once
const holidays = [
  new Date(2025, 6, 4),   // Independence Day
  new Date(2025, 10, 27), // Thanksgiving
  new Date(2025, 10, 28)  // Day after Thanksgiving
];
datePicker.addDisabledDates(holidays, "Public Holiday");

// Enable a previously disabled date
datePicker.removeDisabledDate(new Date(2025, 10, 28));

// Clear all disabled dates
datePicker.clearDisabledDates();

// Check if a date is disabled
if (datePicker.isDateDisabled(new Date(2025, 11, 25))) {
  console.log("This date is disabled");
}

// Get the reason a date is disabled
const reason = datePicker.getDisabledDateReason(new Date(2025, 11, 25));
console.log(reason); // "Christmas Day"
```

#### Disabling Weekdays

```javascript
// Disable weekends (0 = Sunday, 6 = Saturday)
datePicker.disableWeekdays([0, 6]);

// Disable a specific weekday
datePicker.disableWeekday(1); // Disable Mondays

// Enable a previously disabled weekday
datePicker.enableWeekday(0); // Enable Sundays

// Clear all disabled weekdays
datePicker.clearDisabledWeekdays();

// Check if a weekday is disabled
if (datePicker.isWeekdayDisabled(6)) {
  console.log("Saturdays are disabled");
}

// Get all disabled weekdays
const disabledWeekdays = datePicker.getDisabledWeekdays();
console.log(disabledWeekdays); // e.g., [0, 6]
```

#### Disabling Months

```javascript
// Disable summer months (June, July, August)
datePicker.disableMonths([5, 6, 7]); // Note: Months are zero-based (0 = January)

// Disable a specific month
datePicker.disableMonth(11); // Disable December

// Enable a previously disabled month
datePicker.enableMonth(5); // Enable June

// Clear all disabled months
datePicker.clearDisabledMonths();

// Check if a month is disabled
if (datePicker.isMonthDisabled(11)) {
  console.log("December is disabled");
}

// Get all disabled months
const disabledMonths = datePicker.getDisabledMonths();
console.log(disabledMonths); // e.g., [5, 6, 7]
```


#### Usage Example

```javascript
const datePicker = document.querySelector('odyssey-date-picker');

// Listen for date selection changes
datePicker.addEventListener('date-change', (event) => {
  const detail = event.detail;
  
  if (detail.date) {
    // Single date mode
    console.log(`Date selected: ${detail.date}`);
    if (detail.hasEvents) {
      console.log(`Events: ${detail.events.join(', ')}`);
    }
  } else {
    // Range mode
    console.log(`Range: ${detail.rangeStart} to ${detail.rangeEnd}`);
    console.log(`Available dates in range: ${detail.availableDates.length}`);
  }
});

// Example of combining multiple event listeners
['calendar-open', 'calendar-close'].forEach(eventName => {
  datePicker.addEventListener(eventName, () => {
    console.log(`Calendar ${eventName.split('-')[1]}ed`);
  });
});
```

## Implementation Details

### View Modes

The component supports three different view modes:
1. **Calendar View (Days)**: Shows a monthly calendar with individual days
2. **Month View**: Shows a grid of all months for month selection
3. **Year View**: Shows a grid of years for year selection

The component automatically switches between these views based on user interaction.

### Range Selection

When in range mode, the component allows selecting two dates to form a range:
1. First click sets the start date
2. Second click sets the end date
3. The component automatically ensures start date comes before end date

The input displays the range with a separator (e.g., "2025-04-15 - 2025-04-26").

### Direct Text Input

Users can type dates directly in the input field:
- Single dates are parsed according to the specified format
- For ranges, the component intelligently parses text with separators like "-", "to", or "/"
- Invalid input reverts to the previous valid date or clears if no valid date exists

### Event System

The component implements a sophisticated event batching system to prevent duplicate events:
- Multiple state changes within a short time window are batched
- Only a single event is dispatched with comprehensive detail information
- Events include the source of the change (calendar, manual input, API call)

### Touch Gestures

On mobile devices, the component supports touch gestures:
- Swipe down to close the calendar
- Swipe left/right to navigate between months, years, or decades
- Smooth animations provide visual feedback during navigation

## Integration Examples

### JavaScript Frameworks

#### React

```jsx
import React, { useEffect, useRef } from 'react';

// Import the component definition
import '@odyssey/components/date-picker';

function MyDatePickerComponent() {
  const datePickerRef = useRef(null);
  
  useEffect(() => {
    // Access the web component instance
    const datePickerElement = datePickerRef.current;
    
    // Add event listener
    const handleDateChange = (event) => {
      console.log('Selected date:', event.detail.date);
    };
    
    datePickerElement.addEventListener('date-change', handleDateChange);
    
    // Clean up
    return () => {
      datePickerElement.removeEventListener('date-change', handleDateChange);
    };
  }, []);
  
  return (
    <odyssey-date-picker 
      ref={datePickerRef}
      format="MM/dd/yyyy"
      value="2025-04-26"
    />
  );
}
```

```html
document.getElementById('my-form').addEventListener('submit', (event) => {
  event.preventDefault();
  
  const datePicker = document.getElementById('event-date');
  const selectedDate = datePicker.getDate();
  
  if (selectedDate) {
    // Format for form submission
    const formattedDate = datePicker.stateService.formatter.format(selectedDate, 'yyyy-MM-dd');
    
    // Create a hidden input to include in form data
    const hiddenInput = document.createElement('input');
    hiddenInput.type = 'hidden';
    hiddenInput.name = 'eventDate';
    hiddenInput.value = formattedDate;
    
    // Add to form and submit
    event.target.appendChild(hiddenInput);
    event.target.submit();
  }
});
</script>
```
