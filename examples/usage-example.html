Odyssey Components Library Demo

This example demonstrates how to use the packaged components from the Odyssey library.

Odyssey Button Component

Usage Code:

<odyssey-button label="Primary Button" variant="primary"></odyssey-button>

Node Explorer Component

Usage Code:

<node-explorer style="height: 400px; width: 100%;"></node-explorer>

<script>
    // Add nodes programmatically
    const nodeExplorer = document.querySelector('node-explorer');
    
    // After the component is loaded, you can add nodes
    setTimeout(() => {
        if (nodeExplorer) {
            nodeExplorer.addNode({
                id: 'node-1',
                label: 'Example Node 1',
                type: 'process'
            });
            
            nodeExplorer.addNode({
                id: 'node-2',
                label: 'Example Node 2',
                type: 'data'
            });
            
            // Connect nodes
            nodeExplorer.addConnection({
                source: 'node-1',
                target: 'node-2'
            });
        }
    }, 500);
</script>
