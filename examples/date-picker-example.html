Odyssey Date Picker Component

A fully accessible date picker component built using web components. Features include keyboard navigation, multiple themes, event indicators, min/max date restrictions, and internationalization support.

Basic Usage

Default date picker with no options:

<odyssey-date-picker></odyssey-date-picker>

Date Constraints

With min and max date restrictions:

<odyssey-date-picker min-date="2025-04-15" max-date="2025-05-15"> </odyssey-date-picker>

Preset Value

With a preset date:

<odyssey-date-picker value="2025-04-25"></odyssey-date-picker>

States

Disabled date picker:

<odyssey-date-picker disabled></odyssey-date-picker>

Required date picker:

<odyssey-date-picker required></odyssey-date-picker>

Internationalization

French locale with Monday as first day of week:

<odyssey-date-picker locale="fr-FR" first-day-of-week="1"> </odyssey-date-picker>

Events API

Date picker with event indicators:

// Adding events programmatically const datePicker = document.querySelector('#events-picker'); datePicker.addEvents({ '2025-04-22': ['Meeting with team', 'Project deadline'], '2025-04-25': ['Conference call'] });

Theme Variants

Light Theme (Default)

<odyssey-date-picker theme="light"></odyssey-date-picker>

Dark Theme

<odyssey-date-picker theme="dark"></odyssey-date-picker>

Minimal Theme

<odyssey-date-picker theme="minimal"></odyssey-date-picker>

High Contrast Theme

<odyssey-date-picker theme="high-contrast"></odyssey-date-picker>

Public Methods

// Public methods const picker = document.querySelector('#methods-picker'); // Get the selected date const date = picker.getDate(); // Set a date programmatically picker.setDate(new Date(2025, 4, 1)); // Clear the selected date picker.setDate(null); // Open and close the calendar picker.open(); picker.close(); // Set min/max dates picker.setMinDate(new Date(2025, 3, 10)); picker.setMaxDate(new Date(2025, 4, 20)); // Set the locale picker.setLocale('es-ES');
